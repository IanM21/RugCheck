
# Rugcheck Bot

Discord bot for rug checking Solana Shitcoins






## Run Locally

Clone the project

```bash
  git clone https://github.com/IanM21/RugCheck
```

Go to the project directory

```bash
  cd RugCheck
```

Install dependencies

```bash
  npm install
```

Start the bot

```bash
  npm run start
```

Check SPL Token

```bash
  /rugcheck [token address]
```
